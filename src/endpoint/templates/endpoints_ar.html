الرئيسية

نقاط الاتصال البرمجية

جميع واجهات REST API والصفحات المتاحة في المنظومة للربط البرمجي والوصول المباشر

JSON بيانات برمجية
صفحة واجهة HTML
بث بث مباشر WebSocket
صورة ملف صورة

الصفحات والواجهات

GET / صفحة لوحة التحكم الرئيسية GET /counts صفحة لوحة الإحصاء اللحظي GET /analytics صفحة الإحصاءات المسجلة مع تحديد فترة زمنية GET /analytics/daily صفحة إحصائيات الوردية اليومية GET /lost-tracks صفحة المسارات المفقودة GET /track-events صفحة سجل المسارات التفصيلي مع فلاتر متقدمة GET /snapshot/view صفحة البث المباشر للكاميرا GET /health/view صفحة صفحة حالة المنظومة التفصيلية GET /settings/conveyor-roi صفحة إعدادات منطقة الكشف على السير GET /guidelines صفحة تعليمات التشغيل والإرشادات

الإحصاء اللحظي

GET /api/counts JSON بيانات الإحصاء الفوري (مؤكد | قيد المعالجة | مصنّف)
WS /ws/counts/stream بث بث مباشر للتحديثات عبر WebSocket
GET /api/bag-types JSON قائمة أنواع الأكياس مع الصور والأوزان

المسارات والأحداث

GET /api/track-events JSON سجل المسارات مع التصفية والترقيم GET /api/track-events/stats JSON إحصائيات مفصّلة مع التوزيعات البيانية
GET /track-events/{track_id} JSON تفاصيل رحلة مسار محدد (بمعرف المسار)
GET /track-events/by-event/{event_id} JSON تفاصيل مسار بمعرف الحدث الفريد (مُوصى به)
GET /track-events/{track_id}/animation JSON بيانات المحاكاة البصرية للمسار
GET /track-events/visualize/{event_id} صفحة صفحة المحاكاة البصرية بمعرف الحدث (مُوصى به)
GET /track-events/{track_id}/visualize صفحة صفحة المحاكاة البصرية بمعرف المسار
GET /track-events/snapshot/{filename} صورة صورة لحظة فقدان المسار (JPEG)

الكاميرا والصور

GET /snapshot صورة التقاط صورة من الكاميرا (خام أو مع التوضيحات) GET /snapshot/info JSON بيانات وصفية عن آخر لقطة (الأبعاد، التوقيت) GET /snapshot/background صورة صورة الخلفية المرجعية للكشف

منطقة الكشف (ROI)

GET /api/conveyor-roi JSON إعدادات منطقة الكشف الحالية
POST /api/conveyor-roi JSON تحديث إعدادات منطقة الكشف (JSON body)

حالة المنظومة

GET /health JSON فحص حالة الخادم مع التشخيصات
