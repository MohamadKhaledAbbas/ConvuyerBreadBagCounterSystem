الإحصاء اللحظي

متابعة مباشرة لخط الإنتاج مع عرض ثلاثي المستويات: المؤكد، قيد المعالجة، والمصنّف حديثاً.

لوحة المؤشرات

تحليلات شاملة للإنتاج مع إمكانية تحديد الفترة الزمنية وعرض توزيع الأصناف والأوزان.

ملخص الوردية

إحصائيات الوردية الحالية بشكل تلقائي مع حساب فوري لأوقات البدء والانتهاء.

سجل المسارات

تتبع تفصيلي لرحلة كل كيس من لحظة الرصد حتى التصنيف النهائي مع محاكاة بصرية.

البث المباشر

عرض حي للكاميرا مع إطارات الكشف والتعرف. تحديث يدوي أو تلقائي حسب الرغبة.

واجهة المطورين

نقاط اتصال REST API للربط البرمجي مع الأنظمة الخارجية واستخراج البيانات.

نقاط الاتصال البرمجية

GET /api/counts بيانات الإحصاء الفوري (مؤكد | قيد المعالجة | مصنّف)
GET /api/counts/stream بث مباشر للتحديثات عبر SSE
GET /api/bag-types قائمة أنواع الأكياس مع الصور والأوزان
GET /api/track-events سجل المسارات مع التصفية والترقيم
GET /api/track-events/stats إحصائيات مفصّلة مع التوزيعات البيانية
GET /track-events/{track_id} تفاصيل رحلة مسار محدد
GET /track-events/{track_id}/animation بيانات المحاكاة البصرية للمسار
GET /snapshot التقاط صورة من الكاميرا (خام أو مع التوضيحات)
GET /health فحص حالة الخادم
