# Conveyor Bread Bag Counter System v2 - Requirements
# ============================================================================
# Platform-Specific Installation
# ============================================================================

# This project has different dependencies for different platforms.
# Please use the appropriate requirements file:

# For Windows/Development (with NVIDIA GPU):
#   pip install -r requirements-windows.txt

# For RDK X5 Platform (with BPU acceleration):
#   pip3 install -r requirements-rdk.txt --break-system-packages

# ============================================================================
# Quick Install
# ============================================================================

# Windows with GPU:
#   1. Install NVIDIA drivers and CUDA
#   2. pip install -r requirements-windows.txt
#   3. python check_gpu.py  # Verify GPU is detected
#   4. python main.py --source video.mp4

# RDK X5:
#   1. pip3 install -r requirements-rdk.txt --break-system-packages
#   2. python3 -c "from hobot_dnn import pyeasy_dnn; print('BPU OK')"
#   3. python3 main.py --source 0

# ============================================================================
# Key Differences
# ============================================================================

# Windows:
#   - Uses Ultralytics YOLO (.pt models)
#   - Requires PyTorch with CUDA
#   - GPU inference (CUDA)
#   - OpenCV camera/video
#   - Expected FPS: 15-18

# RDK X5:
#   - Uses BPU inference (.bin models)
#   - No PyTorch needed (uses hobot_dnn)
#   - BPU inference (dedicated AI chip)
#   - ROS2 image topics
#   - Expected FPS: 17-21

# ============================================================================
# Troubleshooting
# ============================================================================

# Windows GPU not detected:
#   pip uninstall torch torchvision
#   pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# RDK BPU not available:
#   Check if hobot_dnn is installed (system package)
#   python3 -c "from hobot_dnn import pyeasy_dnn"
